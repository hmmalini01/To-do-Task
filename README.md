# To-do-Task

Project Description
- The To‑Do Task Project is a simple web application designed to help users organize their daily activities.
- It allows users to add tasks, view them in a structured list, and remove tasks once completed.
- The project demonstrates core JavaScript concepts like DOM manipulation, event handling, and dynamic updates to HTML tables or lists.
- It’s a practical way to practice building interactive user interfaces while reinforcing skills in HTML, CSS, and JavaScript.
- The goal is to create a lightweight, user‑friendly tool that improves productivity and showcases your coding abilities.
